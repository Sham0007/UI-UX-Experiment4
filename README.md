# UI-UX-Experiment4

## Aim:
To design a wireframe prototype for a mobile fitness application (GymPro) using Balsamiq, including essential screens such as Splash Screen, Sign Up Screen, Email Verification Screen, Registration Success Screen, and Sign In Screen in order to visualize the user interface structure and user flow before actual development.
## Algorithm:
Step 1:
Start the Balsamiq Wireframe tool.

Step 2:
Create a new project and select mobile screen layout.

Step 3:
Design the Splash/App Intro Screen:

Add the application logo and name (GymPro).

Include illustration related to fitness.

Add buttons for Create New Account and Sign In.

Step 4:
Create the Sign Up Screen:

Add input fields for Full Name, Email, and Password.

Add a Sign Up button.

Provide options for social login (Google, Facebook, etc.).

Add link for existing users to Sign In.

Step 5:
Design the Email Verification Screen:

Display verification message.

Add OTP input boxes.

Provide a Verify Email button.

Step 6:
Create the Registration Successful Screen:

Display confirmation message “Welcome to GymPro”.

Add illustration.

Provide Get Started button.

Step 7:
Design the Sign In Screen:

Add input fields for Username/Email and Password.

Add Remember Password option.

Provide Sign In button.

Step 8:
Add navigation arrows or flow lines to connect the screens showing the user journey.

Step 9:
Review the wireframe to ensure proper layout and usability.

Step 10:
Save and export the wireframe design.
