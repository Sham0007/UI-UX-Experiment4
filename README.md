# UI-UX-Experiment4

## Aim:
To design a wireframe prototype for a mobile fitness application (GymPro) using Balsamiq, including essential screens such as Splash Screen, Sign Up Screen, Email Verification Screen, Registration Success Screen, and Sign In Screen in order to visualize the user interface structure and user flow before actual development.
## Algorithm:
Step 1: Start the Balsamiq Wireframe application.

Step 2: Create a new project and select a mobile screen template.

Step 3: Design the Splash/App Intro screen with the GymPro logo and navigation buttons.

Step 4: Add a Sign Up screen with input fields for Full Name, Email, and Password.

Step 5: Include social login options and a Sign Up button.

Step 6: Design the Email Verification screen with OTP input boxes.

Step 7: Add a Verify Email button to confirm the user account.

Step 8: Create a Registration Successful screen with a welcome message and illustration.

Step 9: Add a Get Started button to proceed to the application.

Step 10: Design the Sign In screen with Username/Email and Password fields.

Step 11: Include Remember Password option and a Sign In button.

Step 12: Connect all screens using arrows to show the user flow.

Step 13: Review the wireframe layout and save the project.
